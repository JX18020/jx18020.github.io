Arrays/ArrayLists

Arrays and array lists are a way to store many values of the same data type. For example, it can store 1000 emails for a newspaper website, 30 student marks for a class, or 10 different models of cars. Each value in an array is called an element, and each position of each element is called an index.

One-Dimensional Arrays

A one-dimensional array is an array which is linear. It only requires one index to specify any element of the array.

Initializing a One-Dimensional Array

There are two main ways to initialize an array. The first way is to declare the array and initialize each element at the same time. The length of the array is also set at the same time when it initialized this way. For example, the following array would have a length of 8. 

    int [] arr = {1, 6, 3, 6, 0, 3, 2, 7};

Another way is to declare the array while also declaring the length of the array. Since an array is an object, the keyword new must be used to declare the array. The following array would have a length of 56. 

    int [] arr = new int [56];

The square brackets ([]) can be before or after the array name. If it is after the array name, it is called C-Style, named after the programming language C. Although it is allowed, c-style is often discouraged and considered bad programming style in Java. Putting the square brackets after the data type is preferred.

Two-Dimensional Arrays

A two-dimensional array is a way to store many values, like in a one-dimensional array, but it can be stored as a matrix. In other words, the values are stored like a table or coordinate grid. Essentially, a two-dimensional array is like an array inside of an array. The two dimensions of the array are represented by a row and a column and each value is stored under each cell.

Initializing a Two-Dimensional Array

A two-dimensional array is initialized in a very similar way to one-dimensional arrays. Like one-dimensional arrays, a two-dimensional array can be created in two ways. It can be created by directly initializing the values in the array. The following array would have 3 rows and 4 columns. 

    int [][] arr = {{1, 2, 3, 4}, {4, 3, 2, 1}, {2, 3, 4, 1}};

The second way is to declare the length of both the row and column. Again, since an array is an object, the new keyword must be used. The following array would have 8 rows and 14 columns. 

    int [][] mat = new int [8][14];

Multi-Dimensional Arrays

A multi-dimensional array is just an array which has more than two dimensions. The initialization process of a multi-dimensional array is the same as with one or two dimensions, just with as many square brackets as there are dimensions.

Length of Arrays

In Java, the length of an array is stored in a final variable, compared to the method for the String class. That means, to access the length of an array, one needs to use .length. 

    int [] arr = new int [10];
    System.out.println(arr.length);

Output: 

    10

However, since it is stored in a final variable (a constant), the length of an array cannot be changed once it is initialized. The only way to change the length of an array is to create a temporary array, transfer the contents of the original array into the temporary array, create a new array with the wanted size, and transfer the contents back into the original array.

Length of Two-Dimensional Arrays

To access the number of rows in a two-dimensional array, one can use .length like as one would for a one-dimensional array. However, to access the number of columns in a two-dimensional array, you have to specify the row first. Here is an example of that. 

    int [][] mat = new int [8][3];
    System.out.println(mat.length);     // the number of rows in the array
    System.out.println(mat[0].length);  // the number of columns in the array
Output: 
    8
    3

Something which is allowed in Java which is not allowed in many programming languages is a ragged array. A ragged array is a two-dimensional array in which the number of columns in each row is not consistent. 

    int [][] mat = {{1, 2, 3}, {3, 2}, {4, 5, 6, 7}};
    System.out.println (mat.length);    // the number of rows in the array
    System.out.println (mat[0].length); // the number of columns in the first row
    System.out.println (mat[1].length); // the number of columns in the second row
    System.out.println (mat[2].length); // the number of columns in the third row
Output: 
    3
    3
    2
    4

The number of rows in the above array is 3. The number of columns in the array are different, with the first having 3 columns, the second having 2, and the third having 4.

Traversing an Array

The indices of an array of length N would range from 0 to N-1. So, if one were to try to access the element at index 5 in an array which has a length of 5, it would throw an ArrayIndexOutOfBoundsException. To access one specific element of an array, one can use square brackets, while specifying the index of the element. 

    int [] arr = {1, 2, 3, 4, 5};

For example, to access the 3rd element of the above array one would need to use arr[2]. To access every element of the array, the easiest way is to use a for loop, which could loop through the indices of the array.

Here is an example of that: 

    int [] arr = {5, 4, 3, 2, 1};

    for (int i = 0 ; i < arr.length ; i++) {
        System.out.println (arr[i]);
    }

Output: 

    5
    4
    3
    2
    1

To traverse an array with two or more dimensions, one needs to use a nested for loop. For example, for a two-dimensional array, you would need to use one nested for loop.

Here is an example of that: 

    int [][] mat = {{3, 4, 5}, {2, 3, 6}, {3, 7, 9}};
    for (int i = 0 ; i < mat.length ; i++) {
        for (int j = 0 ; j < mat[0].length ; j++) {
            System.out.print (mat[i][j] + " ");
        }
        System.out.println();
    }

Output: 

    3 4 5
    2 3 6
    3 7 9

The first loop in the above code accesses the index of the row of the array. The second loop accesses the index of the column. Used together, the for loops access each element of the array.

Using Arrays as Parameters

Normally, when a value is used as a parameter in a method, it is passed by reference. What this means is that a reference to the variable is passed, and whatever is done to that variable in the method does not affect the original variable. However, when using an array as a parameter in a method, it is passed by value. Instead of passing a reference of that array, the array itself is passed, and whatever is done to the array in the method affects the original array. Here is an example of the difference between passing by reference and passing by value. 

    public class PassingTest {
        private int intValue = 4;
        private int [] arrValues = {1, 2, 3, 4};

        public void changeInt(int passedInt) {
            passedInt++;
        }

        public void changeArr(int [] passedArr) {
            for (int i = 0 ; i < passedArr.length ; i++) {
                passedArr[i]++;
            }
        }

        public static void main (String [] args) {
            PassingTest pt = new PassingTest();
            pt.changeInt(pt.intValue);
            pt.changeArr(pt.arrValues);
            System.out.println (pt.intValue);
            System.out.println();
            for (int i = 0 ; i < pt.arrValues.length ; i++) {
                System.out.print (pt.arrValues[i] + " ");
            }
        }
    }

Output: 

    4

    2 3 4 5

As one can see, the int value in the class remained unchanged, but the int values in the array all increased. This is due to the fact that the int value is passed by reference but the array is passed by value.

ArrayLists

An ArrayList is an object and functions similarly to arrays. However, an ArrayList cannot store primitive data types such as an int, double, or char. If one wants to use a primitive data type with an ArrayList, one can use their wrapper classes, such as the Integer class, the Double class or the Character class. It extends the List interface and has a multitude of methods which are useful for manipulating the contents of the array.

Initializing an ArrayList

Like arrays, since an ArrayList is an object, one needs to use the new keyword to create one. An ArrayList also has different constructors, which means one can create an ArrayList with an initial capacity or a default one with an initial capacity of 10. 

    List<String> str = new ArrayList<String>();     // has an initial capacity of 10
    List<String> str = new ArrayList<String>(7);    // sets the initial capacity to 7

In the first ArrayList, the initial capacity of the list is automatically set to 10, while in the second, it is set to 7.

Also, notice how each ArrayList is declared as a List. Although an ArrayList can be declared as 

    ArrayList<String> str = new ArrayList<String>();

it is generally preferred to use 

    List<String> str = new ArrayList<String>();
Length of an ArrayList

Unlike regular arrays, the capacity of an ArrayList can change. The length of an ArrayList can be accessed using the size() method. The size will automatically increase or decrease if one adds or removes elements. When creating an ArrayList with a default constructor, its capacity will automatically be set to 10, and it can be expanded to cut to fit a certain number of elements.

ArrayList Methods

There are many methods which can be used with an ArrayList which make it easier for someone to manipulate the ArrayList and its contents.

Here are a few important ones:

add(E e) - Adds an element to the end of the ArrayList.
add(int index, E element) - Inserts an element into the specified index and shifts the other element down.

clear() - Clears the ArrayList.

get(int index) - Returns the element at the specified index.

indexOf(Object o) - Returns the index of the first occurrence of the specified element if it is found, if it isn't found, it returns -1.
lastIndexOf(Object o) - Returns the index of the last occurrence of the specified element if it found, if it is not found, it returns -1.

remove(int index) - Removes the element at the specified index and shifts the other elements to fill the empty spot.
remove(Object o) - Removes the first occurrence of the specified element and shifts the other elements to fill the empty spot.

set(int index, E element) - Replaces the element at the specified index with the given element.

size() - Returns the length of the ArrayList.
