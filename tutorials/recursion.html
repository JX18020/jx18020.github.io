Recursion

Recursion is essentially calling a method inside of itself. There are 3 main ways to use a recursive method. There is regular recursion, tail recursion, and mutual recursion.

Recursion

A recursive method is one in which there is a base case, and a method call to itself. A problem can be solved recursively if it can be divided into smaller subsections which can be solved. The base case is the smallest subsection in the problem. Here is an example of a recursive method: 

    public int factorial(int n) {
        if (n <= 1)
            return 1;
        return n * factorial(n - 1);
    }

This method returns the factorial of any number. It checks whether the number is less than or equal to 1, where it would return 1. If it is greater than 1, it returns the number multiplied by the method call again using the number minus 1. Here is a tracing of factorial(5): 

    factorial(5) = 5 * factorial(4) = 5 * 4 * 3 * 2 * 1 = returns 120
    factorial(4) = 4 * factorial(3) = 4 * 3 * 2 * 1 = returns 24
    factorial(3) = 3 * factorial(2) = 3 * 2 * 1 = returns 6
    factorial(2) = 2 * factorial(1) = 2 * 1 = returns 2
    factorial(1) = 1 = returns 1

That means factorial(5) returns 120.

Tail Recursion

Tail recursion is when the recursive call is the last line in a method. 
    public int factorialHelper(int n, int product) {
        if (n <= 1)
            return product;
        return factorialHelper(n - 1, n * product);
    }

    public int factorial(int n) {
        return factorialHelper(n, 1);
    }

This is a tail recursive version of the factorial method. As seen in the method, the last line of the method is the recursive call, and no modifications are made to the return statement other than within the parameters. However, to make sure the method works every time, there is a method in which the factorialHelper() method is called with the parameters n and 1.

Here is the tracing of factorial(5) for the tail recursive factorial method: 

    factorial(5) =  factorialHelper(5, 1)
                    factorialHelper(4, 5)
                    factorialHelper(3, 20)
                    factorialHelper(2, 60)
                    factorialHelper(1, 120)
                    return 120;

Thus, factorial(5) is 120.

Mutual Recursion

Mutual recursion is when two or more methods call each other recursively. So, say method A calls method B, method B also calls method A.

Here is an example of mutual recursion: 

    public int A(int a) {
        if (a <= 1)
            return a;
        return (B(a - 2));
    }

    public int B(int b) {
        return b * (A(b));
    }

In this example, method A() calls method B(), and when method B() is called, it calls method A(). This occurs until the parameter in method A() is equal to or less than 0.

Here is the tracing of A(5): 

    A(5)
    =   B(3)
        =   3 * A(3)
                =   3 * B(1)
                        =   3 * 1 * A(1)
                                    =   3 * 1 * 1 = 3

After tracing the method, we can conclude that A(5) returns 3.
